Customization

The file “.cgx” located in the $HOME directory will be read at program start. The following commands might be useful in this context:
  • ”font”
  • ”menu”
  • ”rot”
  • ”view bg”
  • ”view sh”
  • ”wpos”
  • ”wsize”
Example content of a “.cgx” init file: 
valu vinp /home/user/cgx_templates/readinp.fbl
menu readinp(vinp) read vinp
