Verification examples.

The verification examples are simple examples suitable to test distinct features. They can be used to check whether the installation of CalculiX is correct, or to find examples when using a new feature. All verification examples are stored in ccx_2.19.test.tar.bz2. The larger fluid examples can be found in ccx_2.19.fluidtest.tar.bz2, the larger structural examples in ccx_2.19.structest.tar.bz2.
